# Algo Analyzer

Algo Analyzer is an interactive DSA visualization platform that provides step-by-step dry runs, complexity analysis, and visual explanations for Graph, Dynamic Programming, Sorting, and Searching algorithms.

## Features
- Interactive algorithm visualization
- Step-by-step dry runs
- Time and space complexity analysis
- Graph algorithms visualization
- Dynamic Programming explanations
- Sorting and Searching animations
- Beginner-friendly UI

## Tech Stack
- React
- JavaScript
- JSX
- Data Structures & Algorithms

## Purpose
This project was built to help students understand DSA concepts visually and intuitively.
